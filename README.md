### Install & Run frontend Packages

```
cd frontend
npm install
npm start
```

### Install & Run backend Packages

```
cd backend-express
npm install
npm start
```

### Setup frontend .env

- in /frontend , follow `.env.example` to create `.env`

### Setup backend .env

- in /backend-express , follow `.env.example` to create `.env`

### Setup for backend DB connect

```
cd backend-express
npm install
npm run db-migrate
npm run db-seed
```
